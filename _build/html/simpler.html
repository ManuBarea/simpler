The first time the decorated method is called, its result is stored as a pickle file, the next call loads the cached result from the disk. The cached files are used indefinitely unless the seconds lifespan is defined. The cached files are stored at .cached unless otherwise specificed with the directory argument. The cache file identifier is generated from the method name its arguments unless otherwise specified with the identifier argument.
